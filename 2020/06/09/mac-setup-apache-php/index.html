Mac OS X 設定 Apache + PHP

發表於 2020-06-09 | 分類於 Env

要於 Mac 裡開啟 Apache + PHP 的環境其實很簡單
Mac 原本就已安裝好了 Apache 及 PHP 的相關套件，只需要去啟動即可
1.開啟 Apache
先開啟 Terminal 並輸入 sudo su - 切換為 root 身份
輸入 apachectl start 來開啟 Apache (此時為 Apache 預設的設定)
於瀏覽器中開啟 http://localhost 即可看到 It works，這代表 Apache 已正常啟動

因設定檔 /etc/apache2/httpd.conf 為預設的，故其 DocumentRoot 的目錄會在 /Library/WebServer/Documents
可利用 grep DocumentRoot httpd.conf 來查詢

2.設定 Apache 支援 PHP
首先先切換至 Apache 的設定路徑中 /etc/apache2
先將原本的設定檔備份 cp httpd.conf httpd.conf.bak
利用 vi 來編輯設定檔 sudo vi httpd.conf
找到 #LoadModule php5_module libexec/apache2/libphp5.so 後將最前面的註解 # 刪除後存檔離開
於 DocumentRoot 裡將 phpinfo.php 加入 echo '<?php phpinfo(); ?> > phpinfo.php
重新啟動 Apache : apachectl restart
於瀏覽器中開啟 http://localhost/phpinfo.php 可看到 phpinfo 的資訊即代表此 Web Server 已支援 PHP
