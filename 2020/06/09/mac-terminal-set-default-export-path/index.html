Mac Terminal 設定預設 export path

發表於 2020-06-09 | 分類於 Env

因為在公司工作環境中任何對外網路的連線皆需透過 proxy 才能夠正常的連線
而若是在 Terminal 下則需要每次手動去輸入
export http_proxy=http://proxy.domain.name:port
export https_proxy=http://proxy.domain.name:port
之後才能夠在下任何網路的指令時會自動透過 proxy 來做連線
不然就是還要看所下的指令有沒有額外提供設定 proxy 的參數
因此找了一下要怎麼樣在 Terminal 裡不需額外輸入任何指令即能夠自動透過 proxy 連網
vi ~/.bash_profile
在內容登打

之後存檔，如此每次在開啟 Terminal 時即會自動將這兩行指令執行，即不需再為此而煩惱了

若是案照於 Linux 下的習慣多半會編輯在 ~/.bashrc 裡
但因 Mac 系統的設定參照 .bash_profile vs .bashrc 的說法還是需要多再編輯 ~/.bash_profile

如此才能夠正常運作
